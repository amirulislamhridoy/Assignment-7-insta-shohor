How does JavaScript work?

JavaScript is a scripting language.It is interpreter and single thread language. It enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else. Many most modern web browsers support JavaScript and have their JavaScript engines. For example, Google Chrome has its own JavaScript engine called V8. It totally depends on the web-developers how they use JavaScript and for what, because it can be used for several things in web development.

V8 engine is Google's open source high-performance JavaScript and WebAssembly engine. It written in C++. It is used in Chrome, Node.js and among others. V8 engine works in javascript three steps involved in processing the code. (i)Parsing the code (ii)Compiling the code (iii) Executing the code. V8 also has an optimizing compiler jit(just in time) Compiler that produces machine code. when it runs.

Difference among localStorage and sessionStorage?

Local storage size is 10mb.

Does not work for specific tabs.

Data remains after closing browser

The data has to be deleted manually

Session storage size is 5mb

Works for specific tabs

Data doesn't remain after closing browser

The data has to be deleted manually or tab close or browser close
